# Web Development Study Guide

## 1. HTML (HyperText Markup Language)

### 1.1. Fundamentals

-   Document structure: `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`
-   Meta tags and SEO basics
-   Character sets and encoding

### 1.2. Text Elements

-   Headings: `<h1>` to `<h6>`
-   Paragraphs: `<p>`
-   Text formatting: `<strong>`, `<em>`, `<mark>`, `<del>`, etc.
-   Lists: ordered, unordered, description

### 1.3. Links and Navigation

-   Anchor tags: `<a href="">`
-   Internal links and page navigation
-   External links and protocols
-   Email and phone links

### 1.4. Media Elements

-   Images: `<img>` and attributes
-   Audio: `<audio>` and controls
-   Video: `<video>` and embedding
-   iframes and embedding content

### 1.5. Tables

-   Basic structure: `<table>`, `<tr>`, `<td>`, `<th>`
-   Table sections: `<thead>`, `<tbody>`, `<tfoot>`
-   Spanning cells: colspan and rowspan
-   Styling tables

### 1.6. Forms

-   Form structure: `<form>` attributes
-   Input types: text, password, email, number, etc.
-   Checkboxes and radio buttons
-   Select dropdowns
-   Textareas
-   Form validation attributes
-   Buttons and submission

### 1.7. Semantic HTML

-   Structural elements: `<header>`, `<footer>`, `<nav>`, `<main>`, `<section>`, `<article>`
-   Text semantics: `<time>`, `<address>`, `<figure>`, `<figcaption>`
-   Accessibility considerations
-   ARIA roles and attributes

### 1.8. Advanced HTML

-   Custom data attributes
-   SVG integration
-   Canvas element basics
-   HTML templating
